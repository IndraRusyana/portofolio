Tentang Project

Sebuah aplikasi E-Commerce lengkap yang memungkinkan pengguna untuk mencari produk, menambahkan ke keranjang, dan melakukan pembayaran simulasi. Dibangun dengan fokus pada performa dan skalabilitas database.

Latar Belakang

Project ini dibuat untuk menjawab kebutuhan UMKM yang ingin beralih ke digital. Tantangan utamanya adalah bagaimana menangani ribuan data produk dan transaksi secara real-time tanpa membuat server down. Saya memutuskan menggunakan PostgreSQL karena keandalannya dalam menangani relasi data yang kompleks.

Fitur Utama
  • Sistem Autentikasi JWT (Login/Register)
  • Keranjang Belanja Real-time
  • Integrasi Payment Gateway (Midtrans Dummy)
  • Dashboard Admin untuk Manajemen Stok
Tantangan Teknis

Salah satu kesulitan terbesar adalah menangani "Race Condition" saat dua user membeli produk terakhir secara bersamaan. Solusinya, saya menerapkan Database Transaction dan Row Locking di PostgreSQL untuk memastikan konsistensi stok.

Status Pengembangan:
Project ini masih dalam tahap pengembangan fitur "Wishlist" dan perbaikan UI mobile.
